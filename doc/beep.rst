===============
blt::beep
===============

-----------------------
Ring the display's bell
-----------------------


:Author: George A Howlett
:Date:   2012-11-28
:Copyright: 2015 George A. Howlett.
:Version: 4.0
:Manual section: n
:Manual group: BLT Built-In Commands

SYNOPSIS
--------

**blt::beep** ?\ *percent*\ ?

DESCRIPTION
-----------

The **blt::beep** command rings the keyboard bell.  *Percent* is relative
to the base volume of the keyboard bell and can range from -100 to 100
inclusive.

If *percent* is nonnegative then the bell volume is:

  base - [(base * *percent*) / 100] + *percent*

If *percent* is negative then the bell volume is:

  base + [(base * *percent*) / 100]

The default is 50.

KEYWORDS
--------

bell, beep
